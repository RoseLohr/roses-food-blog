# Architecture Decision Records (A-09)

Jede **irreversible** Entscheidung (Datastore, Provider, Protokoll, Mandanten-
modell, Deploy-Form) trägt hier einen ein-seitigen Record: Kontext · Entscheidung
· Konsequenzen · verworfene Alternativen. Aus dem System **wie gebaut**
rekonstruiert. Neue irreversible Entscheidungen ohne ADR fängt die nightly
Fitness-Idee (A-09) / der Review-Prozess ab.
